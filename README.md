# SwipeStitcher

A simple SillyTavern extension that lets you combine the best parts of multiple swipes into one message.

## What it does

Sometimes the AI gives you a few different replies (swipes). Instead of picking just one, SwipeStitcher lets you:

1. Look through all the swipes for a message
2. Highlight the lines or phrases you like most
3. Add them to a combined message
4. (Optional) Polish the combined text to make everything flow together smoothly
5. Save the result as a new swipe

## How to use it

When a message has more than one swipe, a scissors icon appears next to it. Click it to open the stitcher panel.

From there:
- Switch between swipes using the tabs.
- Highlight the text you want and click **Add Selection**.
- Repeat until you have everything you want.
- Click **Polish** to smooth it out, or **Add as Swipe** to save it immediately.

That's it.

## Requirements

- SillyTavern (latest release or staging)
